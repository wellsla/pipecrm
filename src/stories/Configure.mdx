# Welcome to PipeCRM Storybook

## Color Palette

Our design system uses the following colors:

- Primary Dark: `#3A4B59` – Main text and primary UI elements
- Secondary Gray: `#B3B4BC` – Borders and secondary elements
- Background Light: `#FEFFF4` – Page backgrounds
- Accent Red: `#B8284B` – Call-to-action buttons and highlights
- Deep Blue: `#243747` – Headings and emphasis
